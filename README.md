JavaInputTest
=============


Java Development Assessment Test :


A input with characters '1', '0' or '*' 

example 10*00*0


the code must return any possible combination where you can replace '*' with '0' or '1'
 

example:

input > 10*10*0

output > 1001000, 1001010, 1011000, 1011010

 

example:

input > *0

 

output > 00, 10
